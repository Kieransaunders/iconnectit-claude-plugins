The 4-Stage Workflow

The skill operates across a highly dependable four-phase design cycle to maintain absolute visual and engineering quality.

The Briefing Phase

Claude extracts deep context from your project. It queries your brand & offer, primary keywords for localized/global SEO, aesthetic configuration, layout selection (Hero, Bento Features, Testimonials, CTA Band, etc.), and checks if you have DiviTheatre installed for advanced scroll motion features.

HTML Design Preview (Taste Gate)

Before any complex JSON structure is authored, Claude delivers an interactive, modern, ultra-high-fidelity preview-[brand].html layout right inside the Claude UI. This allows you to verify copy rhythms, spacing systems, and visual pacing instantly.

JSON Generation Scripting

Once approved, Claude utilizes an advanced bundled script generator library (divi-builder.js) to assemble semantically perfect block hierarchies including global color mappings, responsive breakpoints, column fractions, and asset-validated image objects.

Validation & Report Card

The output undergoes rigorous automated schema validation (validate.js) ensuring perfect structural health, exact h1/h2 text ratios, mobile-first structural scaling, zero em-dash copy violations, and comprehensive SEO completeness.

Taste & Anti-Slop Dials

Every layout maps configuration to an anti-slop engine. Claude configures three independent dials from 1–10 based on your project goals before rendering structural code.

VARIANCE

Drives structural rhythm. High values inject elegant asymmetric layout splits, offset text columns, and cinematic whitespace block grids over predictable center grids.

MOTION

Drives interaction feedback. Governs element entrances, staggered lists, reveal hooks, and scroll animation integration mapping via Divi 5 native presets.

DENSITY

Drives structural spacing. Balances layout breathing room. High density optimizes technical control rooms with hairline dividers, while low density spaces high-fashion viewports.

Non-Negotiable Guardrails Included
  • 🛡️ Exactly One H1 Per Page: Locked to the primary Hero Headline. No heading level skipping permitted.
  • 🛡️ No Fake AI Elements: Banned section-number eyebrows, decorative dots, scroll indicators, or dummy labels.
  • 🛡️ No Inter/Serif Abuse: Inter is discouraged as a prominent display heading font. Cliché serifs are strictly gatekept.
  • 🛡️ Zero Dash Slop: The structural validator automatically fails if em-dashes (—) or en-dashes (–) are present in page text copy.

Validation & Importing

Every layout delivers an independent set of three production-ready deployment assets in your active workspace:

📦 [brand]-landing-page.json — The core layout library tree.

📦 [brand]-seo-meta.json — Fully matching meta descriptions, slugs, and title tags.

📦 [brand]-schema.json — Clean, automated JSON-LD schema integration for header deployment.

To deploy the file into WordPress, simply navigate to your Divi Library > Portability > Import dashboard, select the generated page layout file, and ensure "Import Presets" is checked to inherit the complete global color and typography styling matrix natively.
