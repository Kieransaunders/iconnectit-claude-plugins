DiviTheatre Motion

DiviTheatre is a free WordPress plugin that adds cinematic scroll animations to Divi 5 — fade-up, stagger, parallax, hero-reveal, and more. It runs on Theatre.js and requires no page builders or shortcodes.

Animations it adds:

  • fade-up — text and sections fade in as they scroll into view
  • stagger — card grids animate in sequence, 100ms apart
  • parallax-scroll — hero images drift as you scroll
  • hero-reveal — choreographed hero: background → headline → body → CTA
  • hover-grow — buttons and cards gently scale on hover
  • scale-in — images and blurbs zoom in from 90%

How to install:

  1. Download the plugin ZIP from your iConnectIT account
  2. WordPress admin → Plugins → Add New → Upload Plugin
  3. Upload the ZIP and click Install Now, then Activate
  4. No configuration needed — it activates automatically on Divi 5 pages

If you select "Not installed but I want it", the generator builds a static page and notes exactly which elements should get animations once you install it.
